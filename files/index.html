What is PDFX
Browser-based PDF toolkit. Everything runs on your device. pdf.js and pdf-lib load once from a CDN when you first open the page — that is the only network request this app ever makes. After that, no data of any kind is sent anywhere. Your files are read directly into browser memory, the same way any desktop app reads a file, and never leave your device.
✓ Zero uploads  ·  Zero server  ·  Zero account
✓ All operations create new files — originals always untouched
✓ Files never leave your device
Memory & Limits (v1)
Per file: under 50MB loads freely  ·  50–100MB shows a warning  ·  over 100MB is blocked
Workspace total: 200MB shows a warning  ·  300MB hard limit
These limits are deliberately conservative for v1 to prevent crashes, and will be revised after stability testing.

Non-Destructive Guarantee
Every tool in PDFX creates a new file. Your original stays in the workspace, clearly marked ORIGINAL. When you download an original it saves as filename_copy.pdf — a different name — so it cannot accidentally overwrite your real file on disk.
✓ Original file on your computer: untouched
✓ Every operation produces a new workspace file
✓ You can always return to the original in the workspace
File Naming Conventions
Manually renamed: downloads exactly as you typed it
Original, unchanged: filename_copy.pdf
One operation applied: filename_watermarked.pdf
Multiple operations, in order: filename_split-p1-3_protected.pdf
Merged file: firstfilename_merged.pdf
Split file: filename_split-p1-3.pdf (includes page range)
Annotated file: filename_annotated.pdf
Storage & Browser Backup
By default, files only exist in JavaScript memory (RAM). If the tab closes, crashes, or refreshes, unsaved work is gone. There is no autosave.
Browser backup (opt-in): stores files in IndexedDB, your browser's local database. Survives tab closes and browser restarts.

Can be lost if you: clear browser data  ·  use private/incognito mode  ·  switch to a different browser  ·  your browser clears storage under memory pressure

This is a safety net, not a guarantee. Download your files to keep them permanently.
Encryption & Security
This app adds no encryption to your files beyond what your browser and operating system already provide.
Files in browser memory: not encrypted by this app
IndexedDB backup: not encrypted by this app
Device-level encryption (FileVault, BitLocker, etc.): outside this app's control — depends on your device setup

What IS encrypted: password-protected output PDFs use real PDF encryption once downloaded. Redacted content is permanently removed from output files.

For sensitive documents: open the app, do your work, download the result immediately, clear the workspace when done. Do not leave sensitive files sitting in the workspace longer than necessary.
Browser Extensions Warning
This app cannot detect, block, or protect against browser extensions. Extensions with "read and write access to all websites" can see everything on this page including any PDF content you are working with.
Extension categories that commonly have broad page access:
Password managers  ·  Screen capture & recording tools  ·  Clipboard managers  ·  Grammar & writing assistants  ·  Ad blockers & privacy tools  ·  Translation extensions

To disable extensions for a session:
Chrome: copy chrome://extensions into your address bar
Firefox: copy about:addons into your address bar
Safari: Safari menu → Settings → Extensions → disable all
Tools Reference
Merge — combine multiple PDFs in any order you set
Split — by page range, every N pages, or each page separately
Reorder — drag/move pages, save as new file
Rotate — per page or all pages, 90°/180°/270°
Pages — delete pages, insert blank pages, duplicate pages
Compress — structural optimization and metadata removal. Note: image recompression is not available in browser-only processing
Watermark — diagonal text overlay, configurable opacity and color
Header/Footer — left/center/right zones supporting {page}, {total}, {date}, {filename} tokens
Page Numbers — multiple positions and format options
Grayscale — rasterizes pages into grayscale JPEG images. Note: may increase file size at high DPI settings
Image→PDF — wraps JPG/PNG/WEBP images into a PDF, each image becomes one page
Extract Text — extracts selectable text only. Does not perform OCR — scanned image PDFs will return no text
PDF→Images — exports each page as a PNG or JPEG file
Redact — draw rectangles over content to permanently black it out in the output file. Always verify redacted areas before sharing
Protect — encrypt with a password using pdf-lib's built-in encryption. Controls print and copy permissions
Unlock — remove password protection. You must know the current password
Annotate — highlights and sticky notes, burned permanently into a new output file
Reader — full-bleed distraction-free reading with column crop/zoom-lock for research papers and multi-column layouts
