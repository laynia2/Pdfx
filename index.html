About PDFX
What is PDFX
PDFX is a browser-based PDF toolkit. Everything runs on your device. When you open this page, pdf.js and pdf-lib are loaded once from a CDN (that is the only network request this app ever makes). After that, no data of any kind is sent anywhere. Your files are read into browser memory directly — the same way any desktop app reads a file — and never leave your device.
Memory & Limits
Per-file limit: 50MB loads freely · 50–100MB shows a warning · 100MB+ is blocked in v1
Workspace limit: 200MB shows a warning · 300MB is the hard limit
Why conservative? This is v1, untested at scale. These limits prevent crashes and will be revised after stability testing.

Non-Destructive Guarantee
Every operation in PDFX creates a new file. Your original is never modified, overwritten, or deleted. It stays in the workspace marked as ORIGINAL. When you download an original, it is saved as filename_copy.pdf so it cannot accidentally overwrite your real file on disk.
✓ Original file on your computer: untouched
✓ Operations always produce a new workspace file
✓ You can always go back to the original in the workspace
File Naming Conventions
Manually renamed: downloads exactly as typed
Untouched original: filename_copy.pdf
One operation: filename_watermarked.pdf
Multiple operations: filename_split-p1-3_protected.pdf (in order applied)
Merged: firstfilename_merged.pdf
Split: filename_split-p1-3.pdf (includes page range)
Annotated: filename_annotated.pdf
Storage & Browser Backup
By default, files only exist in JavaScript memory (RAM). If the tab closes, crashes, or refreshes, unsaved work is gone.
Browser backup (opt-in): stores a copy in IndexedDB — your browser's local database. Survives tab closes and browser restarts. Can be lost if: you clear browser data, use incognito/private mode, switch to a different browser, or your browser frees space under memory pressure. This is a safety net, not a guarantee. Download your files to keep them permanently.
Encryption & Security
This app adds no encryption to your files beyond what your browser and operating system already provide.
What this means:
· Files in browser memory are not encrypted by this app
· IndexedDB backup is not encrypted by this app
· Whether your device encrypts storage (FileVault, BitLocker, etc.) is outside our control

What IS encrypted:
· Password-protected PDF output files use real PDF encryption once downloaded
· Redaction permanently removes content in output files

For sensitive documents: open the app, do your work, download immediately, clear the workspace. Do not leave sensitive files sitting in the workspace longer than necessary.
Browser Extensions Warning
Browser extensions can read the content of any page they have permission to view. Many extensions request "read and write access to all websites." This app cannot detect, block, or protect against extensions you have installed.
Extension categories to be aware of:
· Password managers — often have broad page read access
· Screen capture & recording tools — can capture everything visible
· Clipboard managers — capture anything you copy
· Grammar/writing assistants — typically read full page content
· Translation extensions — read full page content
· Ad blockers & privacy tools — often have the broadest permissions

For maximum privacy, use a browser with extensions disabled:
Chrome: paste chrome://extensions → toggle off
Firefox: paste about:addons → disable all
Safari: Safari menu → Settings → Extensions → disable all
Tools Reference
Merge — combines multiple PDFs in order you specify
Split — by page range, every N pages, or each page separately
Reorder — drag/move pages, saves as new file
Rotate — per-page or all pages, 90°/180°/270°
Pages — delete pages, insert blank pages, duplicate pages
Compress — structural optimization + metadata removal. Note: image recompression is not available in pure browser processing
Watermark — text overlay, configurable opacity/color/position
Header/Footer — left/center/right zones with text, page number, date, or filename tokens
Page Numbers — multiple positions and format options
Grayscale — rasterizes pages (increases file size at high DPI)
Image→PDF — wraps JPG/PNG/WEBP images into a PDF
Extract Text — extracts selectable text (not OCR — won't work on scanned image PDFs)
PDF→Images — exports each page as PNG or JPEG
Redact — permanently blacks out selected areas. Always verify before sharing
Protect — password encryption using pdf-lib (128-bit RC4)
Unlock — removes password (you must know the current password)
Annotate — highlight and sticky notes, burned permanently into output
Reader — clean distraction-free view with column crop/zoom lock
